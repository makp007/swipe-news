# Swipe News

Swipe News is a web application that provides users with the latest news articles from various sources. The application is designed to be user-friendly and responsive, ensuring a seamless experience across devices.

## Features

- **Responsive Design**: The application is built with a mobile-first approach, ensuring it looks great on all devices.
- **User Privacy**: We value your privacy. Please read our [Privacy Policy](privacy.html) to understand how we handle your data.
- **Terms of Use**: Familiarize yourself with our [Terms of Use](tnc.html) to understand the rules and guidelines for using our service.

## Getting Started

To get started with Swipe News, simply open the `index.html` file in your web browser. You can also deploy the application on a web server for broader access.

## Assets

The application includes various assets such as images and icons located in the `assets/img/` directory.

## Dependencies

- [Font Awesome](https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.3/css/all.min.css) for icons.
- [Tailwind CSS](https://cdn.jsdelivr.net/gh/creativetimofficial/tailwind-starter-kit/compiled-tailwind.min.css) for styling.
